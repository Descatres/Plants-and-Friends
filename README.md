# Plants&Friends

## Plants compendium able to save and show plants:
- Names
- Species
- Image of the plants
- Descriptions
## And able to track:
- temperature
- humidity

---

#Testing
- You can run [this script](https://wokwi.com/projects/385292642174231553) to simulate the arduino with the sensors emiting data to your device through MQTT
